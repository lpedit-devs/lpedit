============================
LaTeX Installation GNU/Linux
============================

Basic installation
__________________

It is quite a simple install on linux machines though this will take a few minutes as it installs many languages.

Debian/Ubuntu
^^^^^^^^^^^^^

    .. code-block:: none

        ~$ sudo apt-get install texlive-full
