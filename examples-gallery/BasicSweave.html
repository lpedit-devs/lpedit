Basic Sweave¶

A basic example for Sweave¶

See the Sweave page for more information. This file is included as a template in lpEdit.

%A minimial Sweave template
\documentclass[a4paper]{article}
\usepackage{amsmath,pgf,graphicx,textcomp}
\usepackage[utf8]{inputenc}

\title{A Basic Sweave Example}
\author{Author Name}

\begin{document} 
\maketitle 

\section{Section title} 
We show in this example how to create a matrix of random numbers.
The matrix has $N$ rows and $M$ columns.

% code include example
<<label=chunk1>>=
N <- 4  
M <- 5  
mat <- matrix(rnorm(M*N), N) 
print(mat)
@

\end{document}
