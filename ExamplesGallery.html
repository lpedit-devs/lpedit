Gallery of examples¶

The software editor lpEdit focuses on two languages: R and Python. Reports generated using R make use of Sweave when combining R code and prose. For the Python programming language lpEdit uses Sphinx and reStructuredText to combine code and prose.

The majority of examples in this gallery are written for Python and R, because the code may be run at the same time the report is created, which provides for a highly reproducible framework. Additional languages like Ruby and C/C++ may be used via Sphinx, however there is no simple way to run the code at the time of report generation.
