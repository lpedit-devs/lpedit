LaTeX Installation GNU/Linux
============================

Basic installation
__________________

On OS X in order to install LaTeX there are a couple of options.

Debian/Ubuntu
^^^^^^^^^^^^^

    .. code-block:: none

        ~$ sudo apt-get install texlive-full
